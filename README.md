# Padaria / Supermercado - Landing Page

Uma landing page moderna desenvolvida em **HTML, CSS e JavaScript**, criada para apresentar um supermercado/padaria de forma elegante, destacando sua história, produtos, localização e informações de contato.

## Preview

> Adicione aqui uma imagem ou GIF do projeto.

---

# Funcionalidades

- Header com imagem de destaque
- Menu de navegação
- Seção "Nossa História"
- Carrossel de quitandas
- Carrossel para outras seções do mercado
- Mapa integrado via Google Maps
- Informações de localização
- Contatos e redes sociais
- Footer personalizado

---

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Swiper.js
- Google Maps Embed

# Seções do site

## Header

Apresenta uma imagem principal com destaque para o estabelecimento, juntamente com um menu de navegação e uma chamada principal para o usuário.

---

## Nossa História

Área destinada à apresentação da história da empresa, permitindo transmitir confiança e aproximar os clientes.

---

## Nossas Quitandas

Carrossel desenvolvido com **Swiper.js**, exibindo os principais produtos vendidos pela padaria.

---

## Outras Seções

Outro carrossel destinado à apresentação dos diferentes setores do estabelecimento.

Exemplos:

- Limpeza
- Hortifruti
- Açougue
- Mercearia
- Bebidas

# Biblioteca utilizada

## Swiper.js

O projeto utiliza o **Swiper** para criação dos carrosséis.

CDN utilizada:

```html
<link rel="stylesheet"
href="https://cdn.jsdelivr.net/npm/swiper@12/swiper-bundle.min.css">

<script src="https://cdn.jsdelivr.net/npm/swiper@12/swiper-bundle.min.js"></script>
```
